# rightnow-app
RIGHTNOW is an MVP-focused web app built with **Streamlit** that helps users understand their rights in various  scenarios related to ICE . The app provides:

- **Scenario buttons**: Quickly select situations like “ICE at my door” or “Traffic stop”
- **Language toggle**: Switch between English and Español


---

## Tech Stack

1. **Streamlit (Python)** – Frontend + App Shell
2. **OpenAI API** – For AI-generated structured guidance
3. **Local JSON/Markdown files** – Trusted content pack
4. **GitHub** – Version control and collaboration
5. **Deployment** – Streamlit Community Cloud or Render
